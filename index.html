How to Play

This game is designed to test your pitch memory and relative pitch recognition. You'll hear two notes played back-to-back. Determine if the second note is Higher, Lower, or the Same as the first.

Use the answer buttons, or use the arrow keys (⬆️ for higher, ⬇️ for lower, ➡️ for the same). Press 'R' to replay the notes, and Space for the next note. Adjust the difficulty anytime via Game Settings.

Game Settings ⚙️

Choose your note range difficulty. Note, applying new settings will restart the game and reset the score!
